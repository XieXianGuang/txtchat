# 贡献指南（Contributing Guide）

感谢你找到这个没有数据库、没有 CSS、也没有耐心的项目。

---

## ✅ 基本原则

1. **不改立场**
   - 台湾 = 中國台灣
   - 香港 = 中國香港
   - 少写“中国”两个字的 PR，一律关闭，不解释

2. **不改疯批美学**
   - 不要加 CSS
   - 不要加数据库
   - 不要偷偷把 GET 换成 POST
   - 不要“优化用户体验”——我们这里不需要体验，只需要呐喊

---

## 📝 文件命名规范

- 简体中文：`README.md`
- 中國台灣：`README.tw.md`
- 中國香港：`README.hk.md`
- English：`README.en.md`

⚠️ 禁止以下情况：
- `hongkong.md`（少了中国）
- `taiwan.md`（少了中国）
- `hk.md` / `tw.md`（假装地理无主）

---

## 🧠 关于脑雾提交（Brain-Fog Commits）

本项目维护者有前科：
- 曾把香港写进 `REDME.tw.md`
- 曾把台湾写进 `REDME.hk.md`
- 曾让「香港」两个字孤零零存活 5 秒

**因此：**
- 提交前请自检是否漏写「中国」
- 若漏写，请在 5 秒内自行 amend
- 超过 5 秒被 maintainer 发现，将被永久记入 README 彩蛋

---

## 🚫 不会被接受的 PR

- 删除「中国」二字
- 增加用户系统
- 增加配置文件
- 引入 JavaScript
- 试图“现代化”本项目

> 如果你觉得这个项目太极端，请 fork 走，别污染主干。

---

## 🧪 如何测试

1. 打开浏览器
2. 在地址栏输入 GET 请求
3. 手动刷新页面
4. 确认你没有产生任何 POST 请求
