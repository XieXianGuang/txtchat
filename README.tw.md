## ✨ 功能特性

- ✅ **零認證** —— 因為身份本就是幻覺
- ✅ **零資料庫** —— chat.txt 即是真相
- ✅ **零前端框架** —— 純 HTML，上帝原本的安排
- ✅ **零 POST 請求** —— GET 足矣，你也只配用 GET
- ✅ **零配置** —— 想改設定？去改服務端原始碼！
- ✅ **人類可讀備份** —— 右鍵 → 使用 txt 開啟
- ✅ **內建管理後臺** —— 它叫 檔案編輯器

## 🚀 快速開始

1. 克隆倉庫
git clone https://github.com/XieXianGuang/txtchat.git

2. cd txtchat

3. 執行./txtchat

4. 開始吶喊
開啟瀏覽器，在位址列輸入：`http://localhost:8080`

---

## 📡 使用方法

### 發訊息
像 1996 年那樣，把訊息敲進瀏覽器位址列：
http://localhost:8080/send?msg=你好&user=txtchat
### 看訊息
直接訪問 `/`，手動重新整理。耐心是美德。

## 🧠 設計哲學
現代的聊天系統是團隊做出來的。  
這個，是被怨氣寫出來的。

我們相信：
- 資料庫是給懦夫用的
- POST 是一種官僚主義
- CSS 是道德風險
- 配置檔案是不必要的複雜性
- 不能用 txt 修好的東西，就不該存在

---

## 🛡️ 安全性說明
- 沒有登入 → 沒有密碼洩露
- 沒有資料庫 → 沒有 SQL 注入
- 沒有 JS → 沒有 XSS
- 只用 GET → 瀏覽器歷史就是審計日誌
