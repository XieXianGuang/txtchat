## ✨ 功能特性

- ✅ **零認證** —— 因為身份本就是幻覺
- ✅ **零數據庫** —— chat.txt 即是真相
- ✅ **零前端框架** —— 純 HTML，上帝原本的安排
- ✅ **零 POST 請求** —— GET 足矣，你也只配用 GET
- ✅ **零配置** —— 想改設置？去改服務端源碼！
- ✅ **人類可讀備份** —— 右鍵 → 使用 txt 打開
- ✅ **內置管理後台** —— 它叫 文件編輯器

## 🚀 快速開始

1. 克隆倉庫
git clone https://github.com/XieXianGuang/txtchat.git

2. cd txtchat

3. 運行./txtchat

4. 開始吶喊
打開瀏覽器，在地址欄輸入：`http://localhost:8080`

---

## 📡 使用方法

### 發消息
像 1996 年那樣，把消息敲進瀏覽器地址欄：
http://localhost:8080/send?msg=你好&user=txtchat
### 看消息
直接訪問 `/`，手動刷新。耐心是美德。

## 🧠 設計哲學
現代的聊天系統是團隊做出來的。  
這個，是被怨氣寫出來的。

我們相信：
- 數據庫是給懦夫用的
- POST 是一種官僚主義
- CSS 是道德風險
- 配置文件是不必要的複雜性
- 不能用 txt 修好的東西，就不該存在

---

## 🛡️ 安全性説明
- 沒有登錄 → 沒有密碼泄露
- 沒有數據庫 → 沒有 SQL 注入
- 沒有 JS → 沒有 XSS
- 只用 GET → 瀏覽器歷史就是審計日誌
