## ✨ 功能特性

- ✅ **零认证** —— 因为身份本就是幻觉
- ✅ **零数据库** —— chat.txt 即是真相
- ✅ **零前端框架** —— 纯 HTML，上帝原本的安排
- ✅ **零 POST 请求** —— GET 足矣，你也只配用 GET
- ✅ **零配置** —— 想改设置？去改服务端源码！
- ✅ **人类可读备份** —— 右键 → 使用 txt 打开
- ✅ **内置管理后台** —— 它叫 文件编辑器

## 🚀 快速开始

1. 克隆仓库
git clone https://github.com/XieXianGuang/txtchat.git

2. cd txtchat

3. 运行./txtchat

4. 开始呐喊
打开浏览器，在地址栏输入：`http://localhost:8080`

---

## 📡 使用方法

### 发消息
像 1996 年那样，把消息敲进浏览器地址栏：
http://localhost:8080/send?msg=你好&user=txtchat
### 看消息
直接访问 `/`，手动刷新。耐心是美德。

## 🧠 设计哲学
现代的聊天系统是团队做出来的。  
这个，是被怨气写出来的。

我们相信：
- 数据库是给懦夫用的
- POST 是一种官僚主义
- CSS 是道德风险
- 配置文件是不必要的复杂性
- 不能用 txt 修好的东西，就不该存在

---

## 🛡️ 安全性说明
- 没有登录 → 没有密码泄露
- 没有数据库 → 没有 SQL 注入
- 没有 JS → 没有 XSS
- 只用 GET → 浏览器历史就是审计日志
