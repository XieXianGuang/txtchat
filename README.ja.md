## ✨ 機能と特徴

- ✅ **ゼロ認証** —— アイデンティティなど所詮幻
- ✅ **ゼロDB** —— chat.txtこそが真実
- ✅ **ゼロフロントエンド** —— 純粋なHTML、神の意図通り
- ✅ **ゼロPOST** —— GETで十分。お前にはGETだけで十分だ
- ✅ **ゼロ設定** —— 設定を変えたい？ソースを書き換えろ
- ✅ **人間が読めるバックアップ** —— 右クリック → ダウンロード
- ✅ **内蔵管理画面** —— その名は「テキストエディタ」

---

## 🚀 すぐに始める

```bash
git clone https://github.com/XieXianGuang/txtchat.git
cd txtchat
./txtchat
```

ブラウザで叫ぼう：
`http://localhost:8080`

---

## 📡 使い方

### メッセージ送信
1996年のように、URL欄に叩き込め：
```
http://localhost:8080/send?msg=こんにちは&user=txtchat
```

### メッセージ確認
`/` にアクセスして手動更新。忍耐は美徳。

## 🧠 設計哲学
最近のチャットはチームで作る。
これは**やけくそ**で書いた。

信じていることはこうだ：
- DBは臆病者のための箱
- POSTは役所の書類
- CSSは道徳的ハザード
- 設定ファイルは余計な複雑さ
- txtで直せないものは存在してはならない

---

## 🛡️ セキュリティ

- ログイン不要 → パスワード漏洩なし
- DBなし → SQLインジェクションなし
- JSなし → XSSなし
- GETのみ → ブラウザ履歴が監査ログ
